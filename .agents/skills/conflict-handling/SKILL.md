# Conflict Handling

Use when approved sources disagree or critical values are missing.

Do not silently choose a value from filename, modified time, or completed target
drawing.

Allowed unresolved statuses: `TBD`, `UNVERIFIED`, `CONFLICT`,
`HUMAN_REVIEW_REQUIRED`.

Proposals must be labelled `PROPOSED` and excluded from verified-fact scoring.
