# Evaluation Suite

Phase 0 contains only sanitized synthetic fixtures. Synthetic tests verify
schema parsing, forbidden-label handling, positive bundle construction,
contamination scan, deterministic PDF rendering, PDF readability, and grading
profile arithmetic.

Real-project baselines are blocked until:

- synthetic harness audit passes;
- Codex proxy or approved generator backend passes isolation tests;
- exactly one eligible development project passes blind calibration and audit.
