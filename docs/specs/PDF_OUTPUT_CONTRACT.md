# PDF Output Contract

Required outputs:

- `01_生管課用圖_<工程編號>.pdf`
- `02_鈑金施工圖_<工程編號>.pdf`
- `03_沖孔施工圖_<工程編號>.pdf`

All PDFs must include or accompany project ID, output type, panel/sheet ID,
page number, drawing-model hash, renderer version, and generation run ID.

PDF drafts are never fabrication release, certification, or production approval.
